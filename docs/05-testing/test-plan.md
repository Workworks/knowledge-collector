# 测试计划

## 目标

验证 Knowledge Collector 第一版的主题、采集员、RSS/Atom、HTML 规则、任务、资料库、阅读管理、调度、备份恢复和本地部署闭环。

## 范围

- 单元测试：URL、主题规则、质量评分、TLS 信任配置。
- 集成测试：CRUD、RSS/Atom、HTML、去重、阅读状态、调度、任务恢复、备份恢复。
- 安全测试：SSRF 默认阻断、HTML 清洗、响应大小限制、证书校验不被关闭。
- 部署测试：JDK 17 构建、可执行 JAR、Flyway V1—V8、首页与健康接口。

## 环境

- Java 17
- Maven Wrapper
- H2 文件/内存数据库
- WireMock 固定夹具，不依赖实时网站
- Windows 本地 JAR 冒烟实例

## 准入与退出

- 准入：代码可编译，迁移可校验。
- 退出：全量 `mvn clean verify` 成功；失败和错误均为 0；JAR 冒烟接口返回 200；备份能够恢复并查询文章表。

## 风险

- 实时网站证书、限流和格式会变化，因此自动化测试不依赖外网。
- 动态网页、登录页、验证码和付费墙不属于第一版验收范围。
- 性能结论仅适用于本地单用户规模，不代表分布式或公网服务能力。
