## 风险管理规则

1. 每阶段复核风险概率、影响和新增风险。
2. 高影响风险必须在设计或测试中有明确控制措施。
3. 合规风险由管理员最终确认，技术系统只提供保护机制，不能替代法律判断。
4. 已发生风险转为问题，记录真实现象、影响、临时措施和永久修复。
