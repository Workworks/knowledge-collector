# Stage 8 阶段报告

## 阶段目标

完成资料库搜索与组合筛选、阅读详情、收藏和阅读状态、自定义标签与个人笔记，并在进入阶段功能前提供一条从主题创建到采集员、任务、文章和阅读管理均可正常请求的完整测试任务。

## 已完成

- 新增 `http/stage-8-end-to-end.http`，自动传递主题、采集员、任务和文章 ID。
- local Profile 新增固定数据采集员 Provider，可真实生成任务与文章且不访问互联网。
- 资料库支持搜索标题、作者、摘要、正文、来源和标签。
- 支持主题、采集员、审核状态、阅读状态、收藏、归档、标签、最低质量分和排序组合筛选。
- 阅读状态支持 `UNREAD`、`READ`、`ARCHIVED`、`IGNORED`。
- 支持收藏与取消收藏、归档与恢复、忽略、自定义标签和综合个人笔记。
- Flyway V6 新增标签、文章标签关系、文章笔记和阅读状态组合索引。
- 新增阅读管理 REST API 与标签选项 API。
- 资料库、文章详情、任务列表和任务详情改为结构化交互页面。
- 全站升级视觉系统：统一顶部导航、品牌色、卡片、筛选器、表格、按钮、反馈和响应式布局。
- 页面用语将“采集源”呈现为“采集员”，底层 API 与领域模型保持兼容。

## 主要接口

- `GET /api/v1/articles`：全文搜索和组合筛选
- `GET /api/v1/articles/{id}/reading`
- `PATCH /api/v1/articles/{id}/reading/state`
- `PUT /api/v1/articles/{id}/reading/tags`
- `PUT /api/v1/articles/{id}/reading/note`
- `GET /api/v1/tags`

## 验证

- Stage 8 集成测试完整覆盖主题创建、采集员关联、固定资料采集、主题与质量筛选、收藏、已读、标签、笔记和页面渲染。
- 原有 RSS、Atom、HTML、主题、来源、质量控制和安全测试继续通过。
- 全量测试共 19 个，失败 0、错误 0。

## 已知边界

- 当前全文搜索由 H2 显式查询实现，适用于第一版本地规模；达到性能阈值后切换 Lucene Provider。
- 第一版每篇文章维护一条综合笔记；数据库接口可在后续扩展为多条时间线笔记。
- 标签颜色当前使用默认值，标签独立管理与颜色编辑可在后续增强。
- local 固定采集员仅用于开发和测试，production Profile 不注册。
