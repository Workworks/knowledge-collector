# 用户操作手册

## 基本流程

1. 在“主题”创建分类并填写关键词、排除词。
2. 在“采集员”创建 RSS、Atom、HTML 或本地支持的来源并关联主题。
3. HTML 来源进入规则工作台配置选择器并先执行预览。
4. 点击采集员的“执行采集”，或在“运维”配置固定周期。
5. 在“任务”查看发现数、新增数、重复数、错误和触发方式。
6. 在“资料库”搜索、筛选并打开文章。
7. 在阅读页收藏、标记已读、归档、添加标签和笔记。

## 任务异常

- `HTTP-REQUEST-TIMEOUT`：来源在配置秒数内未完整返回。
- `TASK-TIMEOUT`：任务长期没有心跳，系统已自动释放采集员。
- `TLS-CERTIFICATE-UNTRUSTED`：目标证书链不受本机信任。
- `CRAWL-TASK-ACTIVE`：同一采集员仍有有效任务，避免重复采集。

失败任务可以在详情页点击“重试失败任务”。运维页可以手动回收已经超时的任务。

## 数据备份

进入“运维”，点击“创建备份”。备份默认位于数据目录的 `backups/`。恢复必须停机，并参照部署文档操作。
